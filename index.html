Different types of css

  • Inline css: In inline css we can give style in between tag in a same file and we can target a particular tag for styling.
  • Internal css: In interanl css we can give style in between head tag in a same file and we can target a particular tag for styling.
  • External css: In External css we need to create new file with css extension and need to link file address in between head tags and we can target any tag, class or id.
